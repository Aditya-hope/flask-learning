# Flask Learning

Small Flask apps I build while learning.

## Topics Covered
- Basic Flask routing
- Custom decorators (bold, italic, underline)
- Authentication decorator
- Dynamic URL variables
- Debug mode

## How to Run
1. Clone the repo
   git clone https://github.com/Aditya-hope/flask-learning

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   python hello_flask.py

4. Open browser at
   http://127.0.0.1:5000
